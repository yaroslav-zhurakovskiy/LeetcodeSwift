# LeetcodeSwift
LeetcodeSwift is a wrapper written in Swift langauge around [leetcode.com](http://leetcode.com/) api. It uses a combination of HTTP sessions and REST API to interact with the platform. Look at the tests for mode details.
You can use this project to create your personal tools for leetcode.

Great thanks to leetcode.com, a really awesome website!
